# S-system 家計簿

個人用の家計簿管理アプリケーション。収入と支出を記録し、月次・年次で集計できます。

## 主な機能

✨ **収入・支出の記録**
- 日付、カテゴリ、金額を簡単に記録
- メモ機能で詳細情報も保存可能

📊 **自動集計と期間フィルタリング**
- 今期収入・支出・収支を自動計算
- 前期収支との比較
- 年別・月別でのフィルタリング機能
- 期間情報の自動表示（期間終了までの日数など）

💾 **データ管理**
- SQLiteデータベースで安全に保存
- 設定画面から CSV エクスポート可能（UI 言語に合わせたヘッダー）

🌍 **多通貨表示**
- ¥ JPY（日本円）
- Rp IDR（インドネシアルピア）
- $ USD（米ドル）
- € EUR（ユーロ）
- £ GBP（英ポンド）
- *注: 通貨記号の表示のみで、実際の為替換算は行いません*

📱 **レスポンシブデザイン**
- スマートフォン、タブレット、PCに対応
- シンプルで使いやすいインターフェース

## クイックスタート

### ローカルで実行

```bash
# 依存関係のインストール
pip install -r requirements.txt

# アプリケーションの起動
python app.py
```

ブラウザで `http://localhost:5000` を開いてください。

## デプロイ（Render・無料）

**デプロイ：** [Render](https://render.com) の無料 Web Service に GitHub リポジトリを接続する手順のみです（`gcloud` 不要）。**`DEPLOY.md`** に統一して書いてあります。

## 使い方

### 取引の記録

1. **タイプを選択** - 「収入」または「支出」
2. **金額を入力** - 数字のみ入力
3. **カテゴリを選択** - 給与、食料、交通費など
4. **メモ（任意）** - 詳細を記録したい場合
5. **日付を選択** - デフォルトで今日の日付が入っています
6. **「支出を記録」ボタン** をクリック

### 期間フィルタリング

- 年と月を選択して「フィルター」ボタンをクリック
- 特定の期間の収支を確認できます
- 期間情報ボックスには、現在の期間、前回の期間、期間終了までの日数が表示されます

### 通貨の変更

- 画面右上の通貨セレクターで表示通貨を変更できます
- 表示される通貨記号のみが変わります（為替換算は行われません）

## 技術スタック

- **バックエンド**: Python + Flask
- **データベース**: SQLite
- **フロントエンド**: HTML/CSS/JavaScript
- **デプロイ**: Render / Docker / ローカル（`DEPLOY.md` 参照）

## セキュリティとプライバシー

- データは自分が選んだ環境（PC、Render、自前のクラウドなど）にのみ保存されます
- アプリ自体が第三者にデータを送ることはありません（利用するホスティングのポリシーは各サービスに従います）

## データの永続化について（重要）

**Render の無料 Web Service や Cloud Run など**、コンテナ型ホスティングでは再起動・再デプロイで **SQLite が消える**ことがあります。長期保存が必要なら永続ディスク・外部 DB・バックアップを検討してください。詳細は `DEPLOY.md` を参照してください。

## ライセンス

## サポート

問題が発生した場合は、以下を確認してください：

1. ブラウザのコンソールでエラーメッセージを確認
2. サーバーログを確認
3. データベースファイル `kakeibo.db` が正しく作成されているか確認
